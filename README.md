# Info JV — Site Vitrine

Site web professionnel pour Info JV, entreprise de services informatiques de Julien Villard.

## Stack Technique

- **Next.js 14.2** (App Router)
- **TypeScript**
- **Tailwind CSS 3.4**
- **Framer Motion** pour les animations
- **Material Symbols** pour les icônes

## Design

Design moderne **Bento Grid** avec:
- Fond sombre: `#0C0C0C`
- Cards crème rosé: `#FFF4F3`
- Accent mint: `#53E1A1`
- Font: Poppins

## Installation

```bash
npm install
```

## Développement

```bash
npm run dev
```

Ouvrir [http://localhost:3000](http://localhost:3000) dans votre navigateur.

## Build Production

```bash
npm run build
npm run start
```

## Déploiement

Le site est déployé sur Vercel.
